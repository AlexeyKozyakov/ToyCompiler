# ToyCompiler
<h4>JVM compiler for toy language</h4>
<p>Usage: java &lt;mainClass&gt; &lt;inputFile&gt; &lt;outDir&gt;</p>
<p>Code example:<br>
var x = 5<br>
var i = 1<br>
var res = 1<br>
while i <= x {<br>
&nbsp;&nbsp;&nbsp;&nbsp;res = res * i<br>
&nbsp;&nbsp;&nbsp;&nbsp;i = i + 1<br>
}<br>
print res<br>
</p>
